# Incubyte Automation Assessment

## Tech Stack

- Java
- Playwright
- Maven
- Cucumber BDD
- TestNG
- Page Object Model (POM)

## Scenario Automated

1. Register a new user account
2. Login with created account
3. Verify account overview page
4. Capture and print account balance

## Execution

Run the project using:

mvn clean test

- Login Successful
- Account Balance Captured Successfully
